## Tool contract examples (read this first when adding tools)

Existing tools are Spring `@Component` classes with `@Tool` methods. They are listed ONLY in `com.tepeu.agent.Tools` via `ToolRegistry.register(...)`. Do NOT add constructor deps on `ChatService`.

Pattern (abbreviated from FileTools / ShellTools):

```java
@Component
public class FileTools {
  @Tool(description = "Read a workspace file")
  public String readFile(@ToolParam(description = "relative path") String path) { /* ... */ }
}
```

3. Stop. Do not edit ChatService.

# Tepeu v0.1.0 — Agentic Operating System

> 发布日期: 2026-07-11

---

## 概述

Tepeu v0.1.0 是 Tepeu Agentic OS 的首个公开发布版本。它提供了一个可运行的 Web 工作台，支持多面板布局、Agent 对话（SSE 流式）、Workspace 管理、记忆系统、文件浏览器和终端。

**核心理念：模型是 CPU，Tepeu 是操作系统。**

## 快速开始

```bash
# 方式 1: Docker
docker compose up -d
# 打开 http://localhost:30141

# 方式 2: 本地运行
cd backend && mvn spring-boot:run
cd frontend && npm run dev  # 开发模式（Vite HMR + 代理到 :30141）
```

首次启动会显示设置向导：配置 API Key → 创建工作区 → 进入主界面。

## 已知限制 (v0.1.0)

| # | 限制 | 影响 | 计划 |
|---|------|------|------|
| 1 | 单用户模式，无认证 | 本地机器可用，不支持多用户 | Phase 2 企业版 |
| 2 | FileTools 仅 read-only | Agent 无法创建/修改文件 | 后续 Phase |
| 3 | 机器离线时无法验证 LLM | 需用户提供可访问的 API Key | — |
| 4 | Spring AI 2.0 `ToolCallback` API deprecated | 工具可视化走 deprecated API，未来需迁移 | 跟进 Spring AI 更新 |
| 5 | 无 git 仓库 | `/code-review` diff-based 不可用 | 建议 `git init` |
| 6 | 前端无 toast 层 | 错误消息依赖内联显示 | 后续 Phase |
| 7 | 无 CI/CD | 手动构建和发布 | 后续 Phase |

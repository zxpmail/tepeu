# 延期想法清单（deferred-ideas）

> 项目演进过程中的"暂时不做但值得记下"的想法 parking lot。
> 安装来源：`core/templates/deferred-ideas-template.md` → 用户项目 `.forge/deferred-ideas.md`（`pnpm forge-install` 写入）

---

## 用法

- 想到一个 feature / 改进 / 重构，但当前 Phase 不做 → 写到下面
- 写明：日期 + 想法 + 为何暂缓（成本/优先级/依赖未就绪）
- 定期 review：升级到 EVOLUTION.md / DEV-PLAN.md / 删除

## 模板条目格式

```
- 2026-07-09 **想法标题** — 一句话描述。为何暂缓：xxx。升级路径：xxx。
```

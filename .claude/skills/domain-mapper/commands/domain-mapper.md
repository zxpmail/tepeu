# Command: /domain-mapper

Entry: `/domain-mapper <domain>` or describe what you want to research. **Full workflow → `SKILL.md`**.

## Quick start

| Step | Action |
|------|--------|
| 1 | Define scope — what domain, how deep (L1 snapshot / L2 standard / L3 deep) |
| 2 | Domain snapshot → `domain-map.md` (brands, products, customers, biz model, pain points, social channels, keywords) |
| 3 | (L2/L3) Deep-dive per competitor/entity → `competitor-analysis.md` |
| 4 | (L3) Social/media analysis → `social-analysis.md` |
| 5 | (L3) Synthesis → `domain-map.md` updated with contradictions, roadmap, risks, blind spots |

## Depth levels

| Level | Outputs | Time |
|-------|---------|------|
| **L1 Snapshot** | `domain-map.md` | ~30 min |
| **L2 Standard** | `domain-map.md` + `competitor-analysis.md` | ~1-2 h |
| **L3 Deep** | `domain-map.md` + `competitor-analysis.md` + `social-analysis.md` | ~half day |

## Notes

- Each step **pauses** for user direction — not a fire-and-forget pipeline
- Every claim is source-attributed in the output
- Timestamped: re-run if >30 days old
